# Telegram Bot Bases

Open-source collection of ready-to-use Telegram bot templates.

## Mission

This repository provides reusable Telegram bot bases that developers can use as a foundation for real projects.

## Included Bases

### Basic Bot
- Command handlers
- Logging
- Config system

### SQLite Bot
- User database
- Auto registration
- Statistics

## Planned Bases

- AI Bot
- Economy Bot
- Support Bot
- Shop Bot
- Giveaway Bot
- Subscription Bot
